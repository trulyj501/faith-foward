# QOTD App

The **QOTD App** is designed to provide users with a clean, distraction-free environment to start their day.

## Features
- Daily curated spiritual quotes
- Clean UI with dawn aesthetics
- Notification reminders for morning routines

This project was built to explore how minimalism and technology can intersect with faith.
